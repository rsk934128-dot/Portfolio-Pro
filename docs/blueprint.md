# **App Name**: Portfolio Pro

## Core Features:

- Profile Showcase: Display the user's professional information, including bio, skills, and contact details.
- Project Gallery: Showcase the user's projects with descriptions, images, and links to live sites or repositories.
- Skills Visualization: Present the user's skills in an engaging visual format, such as a chart or tag cloud.
- Certification Display: Showcase certifications with validation URLs, to show learning progress.
- AI-Powered Personalization Tool: Allows users to input some keywords or phrases about what they want to convey about their personal brand, and uses that as a tool to then make suggestions on textual and visual customization to apply to the site.

## Style Guidelines:

- Primary color: Dark blue (#243A73) for professionalism and trust.
- Background color: Light gray (#E0E5EC) for a clean, modern look.
- Accent color: Teal (#488f98) to draw attention to important elements.
- Font pairing: 'Space Grotesk' for headings and 'Inter' for body text (sans-serif).
- Use modern, minimalist icons to represent skills and technologies.
- Clean and structured layout with clear sections for easy navigation.
- Subtle animations and transitions to enhance user experience.
